Le binaire

Comme dirait Mr Bouillot, c'est "un truc avec des 0 et des 1". Plus précisément, le binaire est la base numérique utilisée pas les ordinateurs. Nous utilisons la base 10 (les chiffres de 0 a 9), mais les ordinateurs utilisent que les chiffres 0 et 1. Nous avons appris comment passer du binaire a la base 10, ainsi que l'inverse.

Exemple

010110 en binaire, c'est 22.

Algorithmes de tris

Juste avant les vacances, nous avons travaillé sur les algorithmes de tri. Ce sont des algorithmes qui permettent de trier des objets d'une liste dans un certain ordre. Dans notre cas, nous trions une liste par ordre croissant.


Pour cela nous avons besoin de:
  • Une Liste
  • Un algorithme
  • Une manière de voir combien d'essais ça a pris (de préférence un compteur)
