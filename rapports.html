Le rapport

Mon avis

Tout d'abord, creer un site web a été un projet très intéressant. Non seulement de la perspective du codage, mais aussi voire toutes les options offertes dans l'HTML et le CSS. il y a tellement de façons de modéliser un site. À notre niveau, cela reste assez simple, mais c'était vraiment bien de voir comment il faut s'y prendre pour créer un site web interactif et facile a comprendre.

HTML

Coder en HTML (Hypertext Markup Language), pour moi, était plus facile à apprendre que Python. C'est aussi parce-que HTML a une seule fonction: Creer des sites. Tout de même, j'ai aimé apprendre comment on pouvait formatter du texte, et il y a plusieurs façons:

  • Utiliser "div" comme un "contenaire de texte", cette commande était particulièrement pour ensuite faire le CSS. Nous pouvons ajouter des "div" dans son element parent, ce qui simplifie encore le code.
  • Utiliser la syntaxe "header", "main", "footer". Ces derniers permettent d'obtenir un code clair, surtout avec l'utilisation d'article" a l'intérieur des blocs de textes.
  • Juste utiliser des paragraphes grace à "p", mais cela nous donne un code pas très compréhensible.

Malgré cela, un site juste en HTML n'est pas le plus beau, d'où viens le CSS.

CSS

CSS signifie Cascading style Sheets, donc comme le nom l'indique, c'est la manière dont on change l'apparence de notre site. Tous les sites qu'on voit utilisent CSS. Ce qui était impressionnant c'est de voir à quelle ampleur CSS peut être utilisé. Il y a des gens qui font des sites si impressionnants seulement grace à leur "stylesheet".

Le CSS est facile à apprendre lorsque qu'on parle anglais, puisque beaucoup de lignes de code sont tiré de l'anglais : padding, font, color, background, text align. Je trouve c'est un langage utile à apprendre et facile à maitriser. J'aime bien aussi le fait qu'il n'y a pas beaucoup de limites a ce qu'on peut réaliser avec le CSS.
